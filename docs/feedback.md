# 약 추가 화면

- 一包化의 경우, 굳이 이름 입력 안 해도 될 것 같음.
  이름 입력 하고 싶은 유저에게는 체크박스 체크하면 이름 입력 칸이 나타나도록 하는 건 어떨까?
- 매일/요일지정/간격지정 어느 것이라도 아침/점심/저녁/자기 전 같은 타이밍 입력 중요
- 수량에 소수점 입력이 안 된다. 나 같은 경우 ワーファリン을 아침에 2.5알 먹음.
- 기본적으로 모든 키보드는 다른 곳 터치로 키보드를 닫을 수 있게 해야 UX에 좋지 않을까 싶음.
- 수량 입력 키보드도 닫을 수 없음.
- 약 추가하고 나면 약 상세 화면으로 넘어가는데 앱 이름이 타이틀로 나와 있는 게 부자연스러움
- 상세 화면 넘어간 뒤에 약 모양이랑 색 고른 거로 표시가 되고 한 번 다른 스크린 갔다 오면 그 때서야 업로드 한 사진이 나오는 게 부자연스러움.
- 스케줄을 '매일'로 설정했는데 미지정으로 나옴.
  - 스케줄은 셋 중 하나는 반드시 선택해야 하는 사양이기 때문에 미지정일 수가 없음.
- 중요한 약으로 표시한 것에 대한 아무런 표시가 없음.

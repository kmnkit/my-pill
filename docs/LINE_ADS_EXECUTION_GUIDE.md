# LINE Ads 실행 가이드 — Kusuridoki (くすりどき)

**작성일:** 2026-02-25
**월 예산:** 2万円 (~$130)
**앱 상태:** 미출시 (출시 전 준비)

---

## 현실 체크: 월 2万엔으로 가능한 것

| 항목 | 수치 |
|------|------|
| 일 예산 | ~667円/일 |
| 추정 CPI (LINE) | 200~500円 |
| 월 예상 인스톨 | **40~100건** |
| 학습 기간 | 2~3주 (데이터 축적 필요) |

**솔직한 평가:**
- 2万엔은 LINE Ads "테스트 예산"으로 적합. 대규모 유저 획득에는 부족
- **목적을 "유저 획득"이 아닌 "데이터 수집 + 크리에이티브 검증"으로 설정**하면 효과적
- 어떤 세그먼트/크리에이티브가 반응하는지 파악 → 예산 증액 시 근거 확보
- Apple Search Ads (일 예산 $5~ 가능)와 병행하면 효율 극대화

## Phase 0: 사전 준비 (앱 출시 전 — 지금 시작)

### Step 1: LINE Business ID 생성

1. **LINE Business ID 등록**
   - URL: https://account.line.biz/login
   - "メールアドレスで登録" 또는 "LINEアカウントでログイン"
   - 비즈니스 타입: 個人事業主 또는 法人
   - 업종: 「IT・通信」>「アプリ」

2. **LINE 공식 계정 개설** (광고 집행에 필수)
   - URL: https://www.linebiz.com/jp/entry/
   - 프랜: **コミュニケーションプラン (무료)**
   - 계정명: `くすりどき｜服薬管理アプリ`
   - プロフィール画像: 앱 아이콘 사용
   - 설명문:
     ```
     一包化にも対応した服薬管理アプリ「くすりどき」の公式アカウントです。
     服薬リマインダー・在庫管理・家族の見守り機能で、毎日の服薬習慣をサポートします。
     ```

3. **LINE Ads (LINE広告) 계정 개설**
   - URL: https://admanager.line.biz/
   - LINE Business ID로 로그인
   - 광고 계정 생성:
     - 계정명: `Kusuridoki_JP`
     - 타임존: `Asia/Tokyo`
     - 통화: `JPY`
     - 업종: `コンピュータ・周辺機器・ソフトウェア` > `アプリ`
   - 결제 방법: 크레딧카드 등록

4. **LINE Tag 설치** (앱 내 전환 추적)
   - LINE Ads Manager > 「共有ライブラリ」>「LINE Tag」
   - 앱 인스톨 측정: LINE SDK 연동 또는 3rd party 트래커 (AppsFlyer, Adjust 등)
   - **소규모 예산이므로 3rd party 트래커 없이 시작 가능** — LINE Ads의 기본 앱 인스톨 측정 활용

### Step 2: 앱 스토어 준비 (출시와 동시에 광고 시작하기 위해)

앱 출시 전 완료해야 할 항목:

- [ ] App Store / Google Play 앱 등록 완료
- [ ] 스토어 URL 확보 (LINE Ads 랜딩 페이지로 사용)
- [ ] 광고 소재 제작 완료 (아래 Step 3)
- [ ] 면책 조항 표시 확인: `※ 本アプリは医療機器ではありません`

월 2万엔에서는 **1 캠페인 — 1 광고 그룹 — 2~3 광고**가 최적.

### 왜 케어기버(40~60세) 세그먼트부터?

| 이유 | 설명 |
|------|------|
| 의사결정 능력 | 시니어보다 앱 인스톨/설정 자체가 쉬움 |
| 바이럴 효과 | 본인 설치 후 부모에게도 설치 → 2명 획득 |

| 검색 행동 | "親 薬 管理" 등 능동적 검색 비율 높음 |
| 감정적 소구 유효 | "離れていても安心" 메시지가 강력 |

### 캠페인 설정값

| 항목 | 설정값 | 이유 |

| 항목 | 설정값 |
|------|--------|

## Phase 3: 운영 & 최적화

### 주간 운영 체크리스트

```
매주 월요일 (10분):
□ 주간 소비 금액 확인 (목표: ~5,000円/주)
□ 인스톨 수 / CPI 확인
□ CTR 확인 (목표: 0.3% 이상)
□ 크리에이티브별 성과 비교
□ 월 예산 잔여 확인 → 페이스 조정
```

### KPI 목표 (월 2万엔 기준)

| 지표 | 목표 | 허용 범위 |
|------|------|----------|
| **월 소비** | 20,000円 | 18,000~22,000円 |
| **인스톨 수** | 50건+ | 최소 30건 |
| **CPI** | 400円 이하 | 최대 600円 ※ |
| **CTR** | 0.3%+ | 최소 0.15% |
| **imp** | 30,000+ | - |

※ CPI 600円 초과가 2주 이상 지속되면 크리에이티브 교체 또는 타겟 조정

### 최적화 판단 기준

**1주차 (학습 기간):** 건드리지 않는다. LINE 알고리즘이 최적 배신처를 탐색 중.

**2주차:** 데이터 확인
- 광고①②③ 중 CTR 최하위 → 새 크리에이티브로 교체
- CPI가 500円 이하면 그대로 유지
- CPI가 500円 초과면 타겟 연령 범위 확대 (35~64세)

**3주차 이후:** 월 단위 사이클
- 승리 크리에이티브 유지, 패배 크리에이티브 교체
- 2주마다 최소 1개 새 크리에이티브 투입 (광고 피로 방지)
- 시즌 이벤트 소구 추가 (GW 여행, 敬老の日 등)

### 예산 스케일업 기준

다음 조건 충족 시 예산 증액 검토:

| 조건 | 기준 |
|------|------|
| CPI 안정 | 3주 연속 400円 이하 |
| CTR 양호 | 0.3% 이상 안정 |
| 앱 리텐션 | D7 리텐션 20% 이상 |
| 예산 여력 | 사업 측면에서 증액 가능 |

**증액 단계:**
- 2万円 → 3万円 (50% 증액, 2주 관찰)
- 3万円 → 5만円 (새 세그먼트 추가: 시니어 or 직장인)
- 5万円 → 10만円 (본격 스케일, 복수 캠페인 구조)

## LINE Ads 심사 주의사항

### 심사 통과를 위한 체크리스트

- [ ] **면책 조항 표시**: `※ 本アプリは医療機器ではありません` (이미지 또는 텍스트)
- [ ] **텍스트 비율**: 이미지 내 텍스트 20% 이하
- [ ] **과장 표현 없음**: "ゼロ", "100%", "完璧" 등 결과 보장 표현 불가
- [ ] **최상급 표현 없음**: "No.1", "日本初" 등 (제3자 검증 근거 없으면 NG)
- [ ] **특정 질환명 미사용**: "糖尿病", "高血圧" 등 직접 언급 금지
- [ ] **랜딩 페이지 일치**: 광고 내용과 스토어 페이지 내용 일치
- [ ] **정치/종교/성인 콘텐츠 없음**
- [ ] **저작권 침해 없음**: 스톡 이미지 라이선스 확인

### 심사 리젝 시 대응

| 리젝 사유 | 대응 |
|-----------|------|

# Kusuridoki (くすりどき) 광고 캠페인 크리에이티브 전략

**작성일:** 2026-02-25
**대상 시장:** 일본
**앱 버전:** 1.0.0
**분석 근거:** Consumer Panel (7명, 2026-02-22), PRD, App Store 메타데이터, 일본 약기법/ASO 시장 조사

---

## Executive Summary

Kusuridoki는 일본 복약 관리 앱 시장에서 "一包化 + 가족 견수 + 해외 시차 대응"이라는 독자적 삼각 포지셔닝을 보유하고 있다. 그러나 Consumer Panel에서 **추천 의향 4.9/10, 유료 전환 3.8/10**이라는 낮은 수치가 나왔으며, 핵심 원인은 (1) 프라이버시 불안, (2) 건강 데이터 화면의 광고 노출이다.

따라서 광고 크리에이티브 전략의 최우선 과제는 **"신뢰 구축"**이다. 효능을 과장하는 것이 아니라, "안심감", "가족의 연결", "생활 습관의 지속"을 감정적 핵심 메시지로 삼아야 한다.

## 2. 광고 크리에이티브 방향

### 2-A. App Store 스크린샷 최적화

**현재 상태 분석:**

현재 6장의 스크린샷 시퀀스는 기능 나열형이다. 일본 App Store의 2025-2026 트렌드와 조사 결과를 반영하면 다음 개선이 필요하다.

**핵심 원칙:**
1. **처음 3장이 승부** -- 90%의 사용자가 3번째 스크린샷 이후 스크롤하지 않음
2. **캡션에 검색 키워드 포함** -- Apple 2025 알고리즘이 스크린샷 캡션 텍스트를 인덱싱 (키워드 가시성 +217% 사례)
3. **일본 시장은 정보 밀도 선호** -- 서양식 미니멀리즘보다 "숫자 + 기능 요약"이 전환율 높음
4. **디바이스 프레임 사용** -- iPhone 16 Pro 프레임 내 UI 배치

**개선된 스크린샷 시퀀스 (6장):**

| # | 현재 | 개선안 | 캡션 (JA) | 키워드 삽입 |
|---|------|--------|----------|-----------|
| 1 | 홈 화면 | **가치 제안 + 홈 타임라인** | 「一包化も個別薬も、飲み忘れゼロへ」 | 一包化, 飲み忘れ |

## 3. 약기법 (薬機法) 준수 카피 가이드

### 3-1. Kusuridoki의 법적 위치

Kusuridoki는 **의약품이 아닌 "복약 관리 도구(ツール)"**이다. 따라서 약기법 제66조(허위/과대 광고 금지)와 제68조(미승인 의약품 광고 금지)의 직접 적용 대상은 아니다. 그러나 **앱 광고에서 의약품의 효능/효과를 암시하는 표현**을 사용하면 약기법 위반으로 간주될 수 있다.

### 3-2. 사용 가능한 표현 vs 사용 불가 표현

#### 사용 가능 (OK)

## 5. Consumer Panel 피드백 반영 전략

### 5-1. 프라이버시 우려 대응 (추천 의향 4.9/10 개선)

**광고 크리에이티브에서의 대응:**

1. **스크린샷 6번을 프라이버시 전용으로 변경** (위 2-A 참조)
2. **모든 배너에 자물쇠 아이콘 추가** -- 시각적 안심 신호
3. **동영상 광고에 "データは安全に保護" 자막 포함**
4. **앱 설명문 상단에 프라이버시 선언 이동** (A/B 테스트)

### 5-2. 건강 데이터 화면 광고 문제 대응

**Consumer Panel에서 가장 강한 부정적 반응이 "건강 데이터 화면의 인터스티셜 광고"였다.**

광고 크리에이티브 전략과는 직접 무관하나, **앱 내 광고 배치가 스토어 리뷰와 추천 의향에 직접 영향**하므로 다음을 권장:

- Interstitial 광고는 이미 제거됨 (커밋 60bd6e8 확인)
- 복약 기록/주간 리포트 화면에서는 배너 광고도 최소화 권장
- "Premium = 광고 제거"보다 "Premium = 고급 기능"으로 업셀 포인트 전환

### 5-3. 게이미피케이션 요소 (젊은 층 리텐션)

스트릭 카운터가 P1 백로그에 있으므로, 구현 후 다음 광고 소재 추가:
- 스크린샷에 스트릭 배지 표시
- TikTok/Reels 용 "7일 연속 달성" 숏폼 콘텐츠

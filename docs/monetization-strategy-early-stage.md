# Kusuridoki (くすりどき)

> **작성일**: 2026-02-26
> **개정일**: 2026-02-26
> **작성자**: Stakeholder (CPO 역할)
> **문서 유형**: 전략 지침 (Strategy Mandate)

## Executive Summary

IAP 판매 불가라는 현재 제약은 **결함이 아닌 기회**다. 약 관리 앱의 핵심 가치는 신뢰이며, 일본 시장에서 신뢰는 "무료로 모든 것을 준 다음 나중에 돈을 받는" 방식으로 형성된다. 지금은 수익보다 유저베이스와 습관 형성에 집중해야 한다. 배너 광고는 최소한의 운영 수익을 확보하는 동시에, 프리미엄 전환의 동기("광고 제거")로도 활용된다.

**핵심 원칙**: 지금 돈을 벌려 하지 말고, 나중에 받을 수 있는 권리를 지금 얻어라.

## 2. 수익화 2단계 로드맵

```
Phase A (출시 ~ 6개월+)          Phase B (리텐션 트리거 충족 시)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Launch & Learn                   Premium When Ready
배너 광고만 / 전 기능 무료        배너 광고 + 프리미엄 IAP
데이터 수집 / 신뢰 구축           리텐션 데이터 기반 출시
```

## Phase A: Launch & Learn (출시 ~ 6개월+)

### 목표

유저베이스 구축 + 습관 형성 + 데이터 수집. 수익은 부산물이다.

### 수익 구조

| 수익원 | 역할 |
|--------|------|
| AdMob 배너 광고 | 운영비 일부 충당, "광고 제거" 전환 동기 씨앗 |
| IAP | 비활성화 유지 (kPremiumEnabled = false) |

> **현실적으로 Phase A 광고 수익은 운영비도 못 채운다.** 이를 수용하는 것이 전략이다.

### 배너 광고 정책

**노출 허용 화면 (현재 구현된 위치만):**
- 홈 화면 (`getHomeBannerAd()`)
- 약 목록 화면 (`getMedicationsBannerAd()`)

**절대 금지 위치:**
- 복용 체크 플로우
- 약 상세 화면
- 스케줄 화면
- 준수율(Adherence) 화면
- 설정 화면
- 온보딩 화면
- 모달 / 다이얼로그

**모니터링 기준**: 앱스토어 리뷰에서 광고 불만 2건 이상 확인 시 → 홈 화면 배너 제거 검토.

### Phase A에서 반드시 수집할 데이터

| 지표 | 수집 방법 | 목표값 |
|------|-----------|--------|
| D1 리텐션 | Firebase Analytics | 40%+ |
| D7 리텐션 | Firebase Analytics | 20%+ |
| D30 리텐션 | Firebase Analytics | 10%+ |
| 약 등록 완료율 | 이벤트 트래킹 | 첫 실행 후 70%+ |
| 보호자 연동 시작율 | 이벤트 트래킹 | 20%+ |
| 여행 모드 사용율 | 이벤트 트래킹 | (기준 없음, 탐색) |
| 배너 광고 클릭율 | AdMob 대시보드 | 0.1~0.5% (헬스 앱 평균) |
| 앱스토어 평점 | 모니터링 | 4.0+ 유지 |

## Phase B: Premium When Ready (리텐션 트리거 충족 시)

### Phase A → B 트리거

**아래 조건을 모두 충족할 때만 프리미엄 출시를 시작한다.**

| 트리거 | 기준 |
|--------|------|
| D7 리텐션 | 4주 연속 25%+ |
| D30 리텐션 | 15%+ |
| 월간 활성 복용 체크 유저 | 200명+ |
| 앱스토어 평점 | 4.0+ (리뷰 20개+) |
| 최소 기간 | 출시 후 6개월 |

**트리거 미충족 시**: Phase A를 연장한다. 기간이 길어지는 것은 문제가 아니다. 리텐션 없이 프리미엄을 출시하면 전환율 0에 가까운 수익화가 된다.

### 프리미엄 기능 방침

**핵심 기능 — 영구 무료 (절대 페이월 금지):**
- 약 등록 및 기본 관리
- 알림 / 리마인더
- 복용 체크
- 여행 모드 (최고의 입소문 기능)
- 보호자 연결 1인

**프리미엄 = 편의 기능 추가:**
- 보호자 추가 연결 (2인 이상)
- PDF 리포트 내보내기
- 광고 제거
- 고급 통계 (월별 adherence 트렌드)

### 가격 방침

G2 패널(7명)의 방향성 신호와 경쟁사 기준을 참고한 초기 가이드라인:

| 상품 | 가이드라인 가격 | 비고 |
|------|----------------|------|
| 월 구독 | ¥400~500/월 | 심리적 저항선 ¥500 이하, MediSafe($4.99/월 ≈ ¥750) 대비 저가 포지셔닝 |
| 연간 구독 | ¥2,400~3,600/년 | 월 환산 ¥200~300 → 월 대비 40%+ 할인으로 연간 유도 |

**실제 가격은 유저 베이스 확보 후 A/B 테스트로 결정한다.** 위 수치는 확정값이 아니다.

### 전환 방침

**그랜드파더링 없음. 공정한 가격으로 단순하게.**

복잡한 얼리어답터 특별 가격 구조는 유지 비용과 혼란을 낳는다. 대신:
- 프리미엄 출시 전 충분한 공지 기간 제공
- 무료 플랜은 핵심 기능을 영구적으로 유지 → 강제 이탈 없음
- "광고가 생긴다"는 수준의 자연스러운 다운그레이드

## 3. 금지 사항 (절대 하지 말 것)

| 금지 항목 | 이유 |
|-----------|------|
| 인터스티셜 광고 재도입 | 이미 제거됨. 약 관리 앱에서 전면 광고는 신뢰 파괴 |
| 리워드 광고 도입 | 의료 데이터와 광고 리워드의 결합은 일본 사용자에게 거부감 |
| 복용 체크 / 알림 페이월 | 알림이 없으면 앱의 이유가 없다 |
| 여행 모드 페이월 | 최고의 입소문 기능. 무료로 두어야 한다 |
| 프리미엄 강제 업셀 팝업 | 의료 앱에서 결제 강요는 최악의 경험 |
| 리텐션 트리거 미충족 상태에서 프리미엄 출시 | 기능이 검증되지 않은 상태에서 유료화하면 "돈 내고 쓸 만한 앱이 아니다"라는 인식 고착 |

## 4. 리스크 분석

### 리스크 1: 광고 수익이 너무 낮아 지속 불가능

**가능성**: 높음
**완화책**: 이것을 리스크가 아닌 설계 의도로 수용. Phase A는 수익 창출 단계가 아님. Firebase Spark 플랜 한도 내 운영 유지, 외부 서버 비용 최소화.

### 리스크 2: 리텐션 트리거가 오래 충족되지 않음

**가능성**: 중간
**완화책**: 리텐션 저조의 원인(UX 문제, 기능 부족, 알림 타이밍)을 Phase A 데이터로 진단하고 반복 개선. 트리거 미달 상태에서 프리미엄 출시를 강행하지 않는 것이 핵심.

### 리스크 3: MediSafe가 일본어 강화하여 진입

**가능성**: 중간 (MediSafe는 B2B 제약사 파트너십에 집중 중, 개인 사용자 일본화는 우선순위 낮음)
**완화책**: 속도. Phase B 트리거 달성 이전에 일본 시장에서 "복약 관리 = くすりどき"라는 인식을 심어야 한다. 브랜드가 형성된 후에는 대형 경쟁자도 대체하기 어렵다.

### 리스크 4: Apple IAP 계정 국적 문제 미해결 시

**가능성**: 불확실
**완화책**:
1. Google Play Android를 Phase B 프리미엄 첫 번째 대상으로 설정
2. iOS는 계속 배너 단독 운영 (법적으로 문제 없음)
3. Apple 계정 문제 해결 후 iOS 프리미엄 추가
4. Japan Android 점유율 30%+ 만으로도 의미 있는 초기 수익 확보 가능

### 리스크 5: 배너 광고가 의료 앱 신뢰도에 부정적 영향

**가능성**: 낮음~중간
**완화책**: 광고 위치를 홈 + 약 목록으로 엄격히 제한. 의약품 광고, 경쟁 앱 광고 AdMob 필터링. 앱스토어 리뷰에서 광고 불만 2건+ 시 홈 화면 배너 제거 검토.

## 5. 결론

Kusuridoki의 초기 수익화 전략은 단순하다:

**지금은 돈을 벌 때가 아니다. 지금은 신뢰를 쌓을 때다.**

일본 시장에서, 약 관리 앱 카테고리에서, 고령자와 만성질환자를 타겟으로 하는 앱이 수익을 서두르면 실패한다. 신뢰가 먼저 형성되어야 지불 의향이 생긴다. IAP 비활성화라는 현재 제약은 이 전략을 강제하는 제도적 안전장치 역할을 한다.

Phase A는 데이터를 모으는 시간이다. D7 리텐션 25%, D30 리텐션 15%, 활성 복용 체크 유저 200명 — 이 숫자들이 채워지면 비로소 "유저가 이 앱에 습관을 들였다"는 증거가 된다. 그 시점에서만 프리미엄이 의미를 갖는다.

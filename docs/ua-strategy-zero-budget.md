- **작성일**: 2026-02-26

## 1. ASO (최고 ROI 활동)

가장 적은 시간으로 가장 많은 유기적 다운로드를 만드는 채널.

**JA 키워드 조정**
- 현재 90/100자 → `お薬手帳`(5자) + `高齢者`(3자) 추가, `ADHD`(4자) 제거 → 94자
- `ADHD` 제거 이유: 약 복용 관리 맥락과 연관성 낮음, 실제 검색 의도 불일치

**스크린샷 순서 최적화**
- 1번: 홈 화면 복용 체크 (G2 소비자 패널 4/7 호평)
- Travel Mode 스크린샷 추가 (차별점 시각화)

**분기별 30분 루틴**
- App Store Connect 검색어 성과 확인 → 낮은 성과 키워드 교체
- 리뷰 트렌드 기반 키워드 힌트 수집

참조: `docs/APP_STORE_METADATA.md`

## 2. 콘텐츠 (월 1~2시간)

**X(Twitter) — 주 1트윗**
- 해시태그: `#服薬管理` `#介護` `#旅行`
- 콘텐츠 방향: 앱 홍보가 아닌 사용 팁 중심
- 예: "海外旅行で薬を飲み忘れないコツ 3つ"

**Note.com — 월 1기사**
- 문제 공간 글 작성 (앱 홍보가 아닌 문제 해결 중심)
- 예제 주제:
  - "海外旅行中の薬管理のコツ"
  - "離れて暮らす親の服薬を見守る方法"
  - "時差がある国への長期滞在で薬を管理する方法"

**앱 리뷰 사이트 — 1회 제출 (약 1시간)**
- AppBank, Appliv에 앱 소개 페이지 등록 요청
- 일본어 앱 검색 사용자 유입 경로 확보

## 3. 앱 내 입소문 장치 (이미 구현됨)

별도 마케팅 없이 기존 기능이 신규 유저를 데려오는 구조.

| 기능 | 입소문 메커니즘 |
|------|--------------|
| 보호자 초대 플로우 (QR + 딥링크) | 초대할 때마다 신규 유저 자연 획득 |
| PDF 리포트 공유 (`share_plus`) | 앱 브랜딩 포함 → 의사/약사에게 노출 |
| EN/JA 이중 언어 | 해외 일본인 ↔ 일본 가족 연결 고리 |

## 4. 현실적 MAU 예측 (제로 예산)

| 기간 | MAU | 근거 |
|------|-----|------|
| 1~3개월 | 20~50 | 지인, ASO 유입 시작 |
| 3~6개월 | 50~150 | ASO 복리 성장, 보호자 바이럴 |
| 6~12개월 | 150~500 | 리텐션 강하면 입소문 시작 |
| 2년차 | 500~2,000 | D30 리텐션 15%+ 전제 |

**핵심 지표**: MAU가 아닌 **DAMC (Daily Active Medication Checkers)**.

> 매일 체크하는 유저 1명 > 다운만 하고 떠나는 유저 100명.

약 복용 앱의 본질적 가치는 습관 형성에 있다. DAMC가 리텐션의 실질 지표이며, 이 수치가 높으면 보호자 초대 바이럴도 자연히 발생한다.

## 5. 차별화 메시지 집중 포인트

진짜 차별점 4가지에 메시지를 집중한다. 나머지는 기능이지 차별점이 아니다.

| 차별점 | 메시지 방향 |
|--------|-----------|
| 여행 모드 (타임존 자동 조정) | "海外旅行中でも薬のタイミングを自動調整" |
| 보호자 원격 모니터링 | "離れて暮らす家族の服薬を見守れる" |
| 日英 이중 언어 | "日本語・英語対応 — 海外在住の方にも" |
| 식전/식後 타이밍 | "食前・食後・就寝前、服薬タイミングを管理" |

**一包化(dose pack)에 대한 입장**
- ASO 키워드로는 유지 (검색 의도 일치)
- 마케팅 핵심 메시지로는 사용하지 않는다
- 이유: 一包化는 "관리의 어려움"을 해소하는 수단이므로 "一包化の管理が大変な方へ"는 논리적 모순. 一包化를 쓰는 사람은 이미 관리를 의료진/약사에게 위임한 상태.

## 6. 하지 말 것 (시간 낭비)

인디 개발자의 시간은 유한하다. 효과 없는 채널에 쓰는 1시간은 ASO 개선에 쓸 1시간을 잃는 것.

| 채널 | 하지 않는 이유 |
|------|--------------|
| Reddit | 일본 유저 거의 없음. 일본어 앱 타겟에 미스매치 |
| ProductHunt | 일본어 앱 노출 효과 미미. 영어권 얼리어답터 중심 |
| YouTube 채널 | 제작 비용 대비 ROI 극단적으로 낮음. 채널 성장까지 6개월 이상 필요 |
| Facebook 그룹 | 일본 시니어 타겟에 Facebook 유효하나 그룹 운영 공수가 과도함 |
| 유료 인플루언서 | 예산 없음. 효과 측정도 어려움 |
| Instagram | 약 관리 앱과 플랫폼 성격 불일치. 일본 약/건강 커뮤니티 약함 |
| LINE 공식 계정 | 팔로워 0에서 시작하는 채널을 키우는 데 최소 6개월. 지금 시점에 비효율 |
| 프레스 릴리스 | 인디 앱은 미디어 픽업 거의 없음. 시간 대비 효과 없음 |
